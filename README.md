Project Overview

<span style="color:red">
This project implements a complete **RAG (Retrieval-Augmented Generation) pipeline** designed to gather, process, and provide up-to-date information about universities. The system **scrapes university websites** to extract critical data such as scholarship details, acceptance rates, and other academic statistics. All the scraped data is **stored in a database**, enabling the chatbot to generate accurate and timely answers.  

I personally implemented the **entire backend, database, and frontend**, creating a fully functional **full-stack application**. The RAG pipeline ensures that even for complex queries, the bot can retrieve the most relevant information, combine it intelligently with the LLM, and respond in a human-like manner.  

The motivation behind this project was the difficulty of finding **up-to-date, consolidated university information**, especially for international students. By automating scraping, storage, and retrieval, this system saves users hours of manual research and provides reliable answers almost instantly.
</span>
